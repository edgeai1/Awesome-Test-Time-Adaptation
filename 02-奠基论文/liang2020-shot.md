# SHOT — Do We Really Need to Access the Source Data? Source Hypothesis Transfer

| 项目 | 内容 |
|------|------|
| **作者** | Jian Liang, Dapeng Hu, Jiashi Feng |
| **发表** | ICML 2020 |
| **链接** | [arXiv 2002.08546](https://arxiv.org/abs/2002.08546) |

## 研究背景
无监督域适应（UDA）的经典方法（如DANN、CDAN）都假设可以同时访问源域和目标域数据。然而在实际部署中，源域数据可能由于隐私保护、存储限制或知识产权问题而无法获取。

## 动机 (Motivation)
在不访问源域数据的情况下，仅利用预训练的源模型和无标签的目标域数据，能否实现有效的域适应？SHOT的核心假设是：源域训练好的分类器头已经编码了判别性的类别边界信息，只要将目标域的特征对齐到源分类器期望的分布上即可。

## 核心贡献
1. 首次系统提出**无源域适应**（Source-Free Domain Adaptation, SFDA）的问题设置和解决方案
2. 提出冻结源分类器、仅调整特征提取器的适应策略
3. 结合信息最大化（互信息最大化 + 多样性正则化）和伪标签自训练实现适应

## 方法详解
- **冻结分类器**：保持源域训练好的全连接分类层不变
- **信息最大化**：(1) 个体熵最小化——使每个样本的预测更自信；(2) 全局多样性——确保整体预测分布均匀
- **伪标签自训练**：使用高置信度的模型预测作为伪标签，进一步微调特征提取器

## 实验设置
- **数据集**：Office-31、Office-Home、VisDA-C、DomainNet
- **骨干网络**：ResNet-50/101

## 实验结果
- Office-31上平均准确率达90.1%，超越多数需要源数据的UDA方法
- VisDA-C上达82.9%
- 关键发现：不需要源数据也能做到甚至超越需要源数据的方法

## 局限性与不足
1. 依赖源域分类器的质量
2. 需要目标域足够多数据，不适用于单样本/小批次场景
3. 离线适应，不适用于纯在线TTA场景
4. 伪标签噪声在域偏移大时显著
5. 未考虑持续偏移
