| **发表** | ICCV 2025 |

## 一、研究背景与动机

### 1.1 单模态适应的局限

现有VLM TTA方法有一个根本性局限：它们要么只适应**文本端**（TPT系列优化提示），要么只适应**视觉端**（调BN统计量）。但CLIP是一个**双模态**模型——图文对齐是CLIP的核心。只适应一端而忽略另一端，无法充分利用CLIP的双模态结构。

### 1.2 BATCLIP的核心思想

**同时在线适应视觉编码器和文本编码器**。具体做法是调整两个编码器中LayerNorm层的参数，同时用类原型与文本特征之间的对齐损失来指导适应。

## 二、方法详解

### 2.1 可适应参数选择

CLIP的ViT编码器使用**LayerNorm**（不是BatchNorm）。BATCLIP只更新两个编码器中LayerNorm的仿射参数（$\gamma$ 和 $\beta$）：

- 视觉编码器的LayerNorm参数：$\theta_V = \{\gamma_V, \beta_V\}$
- 文本编码器的LayerNorm参数：$\theta_T = \{\gamma_T, \beta_T\}$

参数量约占模型总参数的<0.5%。

### 2.2 在线类原型维护

对每个类别 $k$ 维护一个**动态类原型** $\mathbf{c}_k$：

$$\mathbf{c}_k^{(t)} = (1-\alpha) \cdot \mathbf{c}_k^{(t-1)} + \alpha \cdot \bar{\mathbf{z}}_k^{(t)}$$

其中 $\bar{\mathbf{z}}_k^{(t)}$ 是时间步 $t$ 被预测为类 $k$ 的高置信度样本的平均视觉嵌入。

初始原型用CLIP的文本嵌入初始化：$\mathbf{c}_k^{(0)} = \mathbf{t}_k$。

### 2.3 双模态对齐损失

**目标1：类原型-文本对齐**
最大化每个类原型与对应文本嵌入的相似度：

$$\mathcal{L}_{align} = -\frac{1}{C} \sum_{k=1}^C \text{sim}(\mathbf{c}_k, \mathbf{t}_k)$$

**目标2：类原型可分离性**
最大化不同类原型之间的距离：

$$\mathcal{L}_{sep} = \frac{1}{C(C-1)} \sum_{i \neq j} \max(0, m - \text{dist}(\mathbf{c}_i, \mathbf{c}_j))$$

其中 $m$ 是margin超参数。

**总损失**：

$$\mathcal{L} = \mathcal{L}_{align} + \mu \cdot \mathcal{L}_{sep}$$

### 2.4 流程图

```
测试样本流 x₁, x₂, x₃, ...
    │
    ├──→ CLIP视觉编码器(可适应LN) → 视觉嵌入 z
    │
    ├──→ CLIP文本编码器(可适应LN) → 文本嵌入 {t₁,...,tᶜ}
    │
    ├──→ 预测 → 高置信度? → 更新对应类原型 cₖ
    │
    ├──→ 计算对齐损失 + 可分离性损失
    │         │
    │    反向传播 → 同时更新视觉LN和文本LN参数
    │
    └──→ 输出预测
```

## 三、实验设置

- **骨干**：CLIP ViT-B/16, ViT-L/14
- **数据集**：ImageNet-C（15种腐蚀×5级）、域泛化数据集（PACS、VLCS、OfficeHome）
- **基线**：Tent、TPT、TDA、ZERO、CoTTA
- **适应模式**：**在线**（不重置，持续适应）

## 四、实验结果

### 4.1 ImageNet-C（在线设置）

| 方法 | Level 5平均Acc | 适应模态 |
|------|:----------:|:------:|
| CLIP零样本 | 22.7 | — |
| Tent | 28.3 | 视觉BN |
| TPT | 34.2 | 文本提示 |
| TDA | 33.8 | 缓存 |
| **BATCLIP** | **37.1** | **视觉LN+文本LN** |

双模态适应比单模态适应高约3-5%。

### 4.2 域泛化数据集

| 方法 | PACS | VLCS | OfficeHome | 平均 |
|------|:---:|:---:|:---------:|:---:|
| CLIP | 94.1 | 79.2 | 79.5 | 84.3 |
| TPT | 95.3 | 80.8 | 81.2 | 85.8 |
| **BATCLIP** | **96.1** | **82.3** | **82.8** | **87.1** |

### 4.3 消融实验

| 适应模态 | ImageNet-C Acc |
|---------|:----------:|
| 无适应 | 22.7 |
| 仅视觉LN | 30.5 |
| 仅文本LN | 32.1 |
| **视觉LN + 文本LN** | **37.1** |

两个模态的适应是互补的，组合效果远超单独适应。

## 五、局限性与不足

1. **双模态更新增加不稳定性**：同时更新两个编码器的参数空间更大，崩溃风险更高
2. **类原型的准确性**：依赖于初始预测质量——如果CLIP在目标域上初始预测很差，原型会被错误更新
3. **在线模式的顺序敏感性**：测试样本的到达顺序影响原型和参数更新
4. **LayerNorm适应的理论基础**：为什么调LN仿射参数有效，缺乏理论解释
5. **可分离性损失中的margin选择**：$m$ 值需要手动设定

## 六、核心价值

BATCLIP是首个系统性地**同时适应CLIP两个编码器**的TTA方法，证明了双模态适应的互补效果。其"类原型对齐"的思想为后续VLM在线适应方法提供了重要参考。
