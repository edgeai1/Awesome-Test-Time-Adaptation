## 一、研究背景与动机

### 1.1 现有无训练方法的盲区

TDA、ZERO等无训练方法使用预测**熵**作为唯一的视图质量指标：低熵=可靠，高熵=不可靠。但SSG发现这个准则忽略了一些关键的**可泛化因素**：

- **形状 (Shape)**：物体的形状是泛化性最强的特征——一只猫无论是照片、素描还是油画，形状都是相似的
- **风格 (Style)**：纹理、颜色、光照等风格因素是导致域偏移的主要原因——同一只猫在不同光照/背景下风格完全不同

好的模型应该**对形状敏感、对风格不敏感**。但现有方法没有利用这一先验。

### 1.2 SSG的核心思想

通过主动**腐蚀**测试图像的形状和风格，然后观察预测的变化：
- 如果腐蚀形状后预测变化大 → 预测依赖形状（好！）
- 如果腐蚀风格后预测变化大 → 预测依赖风格（不好！）

用这种**预测扰动差异 (PPD)** 来衡量视图的可靠性。

## 二、方法详解

### 2.1 形状腐蚀

对输入图像应用会破坏形状但保留风格的变换：
- **局部遮挡**：随机遮挡图像的关键区域
- **空间打乱**：将图像分成patch后随机排列
- **边缘模糊**：高斯模糊消除物体轮廓

形状腐蚀后的图像记为 $x_{shape-corrupt}$。

### 2.2 风格腐蚀

对输入图像应用改变风格但保留形状的变换：
- **颜色反转/抖动**
- **风格迁移**（简单的统计量迁移）
- **对比度/亮度极端变化**

风格腐蚀后的图像记为 $x_{style-corrupt}$。

### 2.3 预测扰动差异 (PPD)

$$\text{PPD}_{shape}(x) = D_{KL}(p(y|x) \| p(y|x_{shape-corrupt}))$$
$$\text{PPD}_{style}(x) = D_{KL}(p(y|x) \| p(y|x_{style-corrupt}))$$

**可靠性分数**：

$$\text{Score}(x) = \text{PPD}_{shape}(x) - \gamma \cdot \text{PPD}_{style}(x)$$

- PPD_shape高 → 预测依赖形状 → 好（加分）
- PPD_style高 → 预测依赖风格 → 不好（扣分）

### 2.4 PPD引导的缓存更新

与TDA类似维护缓存，但用PPD分数而非熵来决定入库条件：

- 高Score的样本 → 加入缓存（形状依赖性强）
- 低Score的样本 → 不入库（风格依赖性强）

### 2.5 流程图

```
测试样本 x
    │
    ├──→ CLIP预测 p(y|x)
    │
    ├──→ 形状腐蚀 → x_shape → CLIP预测 → PPD_shape = KL(p‖p_shape)
    │
    ├──→ 风格腐蚀 → x_style → CLIP预测 → PPD_style = KL(p‖p_style)
    │
    ├──→ Score = PPD_shape - γ·PPD_style
    │
    ├──→ 高Score → 可靠样本 → 加入缓存 → 缓存辅助预测
    │
    └──→ 最终预测
```

## 三、实验结果

### 3.1 OOD分类

| 方法 | ImageNet-A | -R | -Sketch | OOD平均 |
|------|:---------:|:--:|:------:|:------:|
| TDA | 54.2 | 77.3 | 48.1 | 59.9 |
| ZERO | 55.6 | 77.4 | 48.1 | 60.4 |
| **SSG** | **56.8** | **78.2** | **49.5** | **61.5** |

SSG比TDA高+1.6%，比ZERO高+1.1%。

### 3.2 效率对比

| 方法 | 每样本时间(ms) | 反向传播 |
|------|:----------:|:------:|
| TPT | 320 | ✅ |
| TDA | 8 | ❌ |
| **SSG** | **24** | **❌** |

SSG需要额外的腐蚀和前向传播（约3x于TDA），但仍远快于TPT（约13x）。

### 3.3 消融实验

| 指导信号 | ImageNet-A |
|---------|:---------:|
| 仅熵（=TDA） | 54.2 |
| 仅PPD_shape | 55.9 |
| 仅PPD_style（取反） | 55.1 |
| **PPD_shape - γ·PPD_style** | **56.8** |

形状和风格双重指导最有效。

## 四、局限性与不足

1. **腐蚀操作的设计是手动的**：什么变换算"形状腐蚀"、什么算"风格腐蚀"需要先验知识
2. **增加3-4次前向传播**：虽比TPT快，但比TDA/ZERO慢约3倍
3. **形状/风格解耦不完美**：某些变换可能同时影响形状和风格
4. **对于某些领域（如纹理分类），形状先验不适用**——DTD数据集上分类靠纹理（风格）而非形状

## 五、核心价值

SSG引入了**领域知识先验**（形状不变性/风格可变性）来指导无训练TTA的样本选择。这是首次将**表征学习中的形状偏好（shape bias）理论**应用于测试时适应，开辟了"先验引导的TTA"新方向。
