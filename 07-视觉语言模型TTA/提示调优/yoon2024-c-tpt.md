## 一、研究背景与动机

### 1.1 TPT的校准问题

TPT通过熵最小化优化提示，确实能提升准确率。但作者发现一个被忽视的严重问题：**校准性能下降**。

**什么是校准（Calibration）？** 一个校准良好的模型，当它说"我90%确信这是一只猫"时，它的预测应该确实有90%的概率是正确的。校准差的模型可能以99%的置信度给出错误预测——这在医疗诊断、自动驾驶等安全关键应用中是致命的。

TPT的熵最小化本质上是"逼"模型变得更自信，但**自信不等于正确**。这导致了预测概率分布过于尖锐（过度自信），ECE（Expected Calibration Error）显著增大。

### 1.2 关键发现

作者通过大量实验发现：文本特征在CLIP特征空间中的**离散度**（dispersion）与模型的校准性呈强正相关。

- **离散度低**：不同类别的文本嵌入在特征空间中挤在一起 → 类别难以区分 → 校准差
- **离散度高**：不同类别的文本嵌入分散均匀 → 类别界限清晰 → 校准好

## 二、方法详解

### 2.1 平均文本特征离散度 (ATFD)

定义ATFD为所有类别对之间文本嵌入的平均余弦距离：

$$\text{ATFD} = \frac{2}{C(C-1)} \sum_{i < j} (1 - \text{sim}(\mathbf{t}_i, \mathbf{t}_j))$$

其中 $C$ 是类别数，$\mathbf{t}_i = f_T([v_1, ..., v_M, c_i])$ 是第 $i$ 类的文本嵌入。

ATFD越大 → 文本嵌入越分散 → 校准越好。

### 2.2 C-TPT优化目标

在TPT的熵最小化基础上，增加ATFD最大化项：

$$\mathcal{L} = H(\bar{p}) - \lambda \cdot \text{ATFD}$$

其中 $H(\bar{p})$ 是TPT原有的平均预测熵（最小化），$\text{ATFD}$ 是文本特征离散度（最大化），$\lambda$ 是平衡系数。

### 2.3 为什么ATFD有效？

直觉解释：
- 熵最小化让模型对每个样本的预测更自信 → 容易过度自信
- ATFD最大化让不同类别的文本表征更分散 → 类别决策边界更清晰
- 两者结合：模型在"该自信的时候自信，该不确定的时候不确定"

### 2.4 计算流程

```
TPT流程 (与TPT相同)
    ├── 增强 → 选择 → 计算 H(p̄)
    │
    ├── 计算所有类别文本嵌入 {t₁, ..., tᶜ}
    │         │
    │    计算两两余弦距离 → ATFD
    │
    └── 总损失 L = H(p̄) - λ·ATFD → 更新提示
```

### 2.5 计算复杂度分析

ATFD计算需要 $O(C^2)$ 次余弦相似度计算。对于类别数 $C$ 较小时（如ImageNet-A有200类），这是可以接受的。但对于ImageNet的1000类，需要约50万次配对计算。

实际实现中，ATFD的计算可以高效批量化，开销远小于TPT本身的N次前向传播。

## 三、实验设置

### 3.1 关键超参数
- 继承TPT的所有设置（N=63, Top 10%选择, 1步优化）
- ATFD系数 $\lambda$：0.5
- 骨干：CLIP ViT-B/16

### 3.2 评估指标
- **准确率 (Accuracy)**：标准分类准确率
- **ECE (Expected Calibration Error)**：期望校准误差，越低越好
- **MCE (Maximum Calibration Error)**：最大校准误差
- **AECE (Adaptive ECE)**：自适应分箱的ECE

## 四、实验结果

### 4.1 准确率与校准的联合评估

| 方法 | ImageNet-A Acc↑ | ImageNet-A ECE↓ |
|------|:---------:|:--------:|
| CLIP零样本 | 47.87 | 5.2% |
| TPT | 54.77 | **12.8%** ← 校准严重恶化 |
| **C-TPT** | **55.1** | **4.1%** ← 校准显著改善 |

关键发现：
- TPT的ECE从5.2%恶化到12.8%（+7.6%）—— 校准大幅下降
- C-TPT的ECE为4.1%，甚至优于零样本CLIP
- C-TPT在准确率上与TPT持平（55.1% vs 54.77%）

### 4.2 多数据集校准改善

| 数据集 | TPT ECE | C-TPT ECE | 改善比例 |
|-------|:------:|:--------:|:------:|
| ImageNet-A | 12.8% | 4.1% | -68% |
| ImageNet-V2 | 8.3% | 3.7% | -55% |
| ImageNet-R | 6.1% | 2.9% | -52% |
| ImageNet-Sketch | 9.5% | 4.5% | -53% |

C-TPT在所有数据集上将ECE降低了约50-68%。

### 4.3 消融实验

**ATFD系数λ的影响**：

| λ值 | Accuracy | ECE |
|:---:|:-------:|:---:|
| 0（= TPT） | 54.77 | 12.8% |
| 0.1 | 55.0 | 8.2% |
| **0.5** | **55.1** | **4.1%** |
| 1.0 | 54.3 | 3.8% |
| 5.0 | 50.2 | 2.9% |

λ=0.5是最佳平衡点。λ太大虽然校准好，但准确率下降。

## 五、局限性与不足

1. **继承TPT的速度问题**：仍需64x前向+1x反向传播
2. **ATFD计算在大类别数时开销增大**：$O(C^2)$ 复杂度，ImageNet-1K需50万次配对
3. **对语义相近的类别可能有害**：如"金毛犬"和"拉布拉多犬"语义很近，强制它们的文本嵌入分散可能损害判别性
4. **λ需要手动调节**：最优λ值可能因任务而异
5. **仅在文本端做校准**：视觉特征端的校准问题未解决

## 六、核心价值

C-TPT的核心贡献是**将校准性作为VLM TTA的重要评估维度**。在此之前，VLM TTA领域几乎只看准确率。C-TPT揭示了一个重要警告：

> 准确率的提升不应以校准性的牺牲为代价。一个过度自信但经常犯错的模型，比一个适度自信的模型更危险。
