## 一、研究背景

### 1.1 校准问题回顾

C-TPT首次揭示了TPT的校准退化问题，并用ATFD（平均文本特征离散度）来改善。然而，C-TPT的ATFD只是最大化文本特征之间的**平均余弦距离**，这不能保证所有类别对之间的距离都足够大——可能存在某些类别对仍然高度重叠。

### 1.2 O-TPT的洞见

O-TPT进一步发现：**角度分离（angular separation）**是影响校准的关键因素。在CLIP的归一化特征空间（单位超球面）上，类别文本嵌入之间的角度越大 → 决策边界越清晰 → 校准越好。

关键区别：ATFD最大化**平均**距离，O-TPT最大化**最小**成对角距离——确保没有任何两个类别过于接近。

## 二、方法详解

### 2.1 正交性约束

对C个类别的归一化文本嵌入 $\{\hat{\mathbf{t}}_1, ..., \hat{\mathbf{t}}_C\}$（$\hat{\mathbf{t}}_k = \mathbf{t}_k / \|\mathbf{t}_k\|$），构造Gram矩阵：

$$G_{ij} = \hat{\mathbf{t}}_i^\top \hat{\mathbf{t}}_j$$

理想情况下（完全正交），$G$ 应该是单位矩阵 $I$。正交性损失定义为：

$$\mathcal{L}_{orth} = \|G - I\|_F^2 = \sum_{i \neq j} (\hat{\mathbf{t}}_i^\top \hat{\mathbf{t}}_j)^2$$

这个损失惩罚任何两个类别文本嵌入之间的余弦相似度——越接近0（正交）越好。

### 2.2 完整优化目标

$$\mathcal{L} = \underbrace{H(\bar{p})}_{\text{TPT的熵最小化}} + \lambda \underbrace{\|G - I\|_F^2}_{\text{正交性约束}}$$

### 2.3 O-TPT vs C-TPT 的关键区别

| 维度 | C-TPT (ATFD) | O-TPT (正交性) |
|------|-------------|--------------|
| 目标 | 最大化平均余弦距离 | 最小化Gram矩阵与单位矩阵的距离 |
| 优化的是什么 | 平均值 | 最小值（最差情况） |
| 语义保护 | 弱（可能拉开语义相近的类） | 强（正交≠反义，只是独立） |
| 计算复杂度 | $O(C^2)$ | $O(C^2)$（相同） |

### 2.4 流程图

```
TPT基础流程（增强→选择→熵最小化）
    │
    ├──→ 计算文本嵌入Gram矩阵 G
    │
    ├──→ 正交性损失 = ‖G - I‖²_F
    │
    └──→ 总损失 = H(p̄) + λ·‖G - I‖²_F → 更新提示
```

## 三、实验设置

### 3.1 关键配置
- **骨干**：CLIP ViT-B/16, ViT-L/14
- **数据集**：ImageNet变体、细粒度数据集、**医学数据集**（ISIC皮肤病变、PathMNIST病理学）
- **评估指标**：Accuracy、ECE、MCE、AECE
- **基线**：TPT、C-TPT、CLIP零样本

### 3.2 超参数
- $\lambda$：0.3-0.8（在不同数据集上调优）
- 其余继承TPT设置

## 四、实验结果

### 4.1 校准性能（CLIP ViT-B/16）

| 方法 | ImageNet-A Acc↑ | ECE↓ | MCE↓ |
|------|:---------:|:---:|:---:|
| CLIP零样本 | 47.87 | 5.2% | 18.3% |
| TPT | 54.77 | 12.8% | 35.2% |
| C-TPT | 55.1 | 4.1% | 14.5% |
| **O-TPT** | **55.3** | **3.4%** | **11.8%** |

- O-TPT的ECE比C-TPT进一步降低约17%（4.1%→3.4%）
- MCE降低约19%（14.5%→11.8%）
- 准确率略高（55.3% vs 55.1%）

### 4.2 聚合校准误差（多数据集平均）

| 方法 | 平均ECE↓ |
|------|:------:|
| TPT | 10.7% |
| C-TPT | 4.8% |
| **O-TPT** | **3.6%** |

O-TPT在所有数据集上一致降低校准误差。

### 4.3 医学数据集（ISIC皮肤病变分类）

| 方法 | Accuracy | ECE |
|------|:-------:|:---:|
| CLIP零样本 | 34.2 | 8.1% |
| TPT | 38.1 | 15.3% |
| C-TPT | 37.8 | 6.2% |
| **O-TPT** | **39.2** | **4.8%** |

在医学场景中，O-TPT不仅准确率最高，校准也最好——这对临床决策至关重要。

### 4.4 消融实验

**不同正交约束方式的对比**：

| 约束方式 | ECE |
|---------|:---:|
| 无约束（TPT） | 12.8% |
| ATFD最大化（C-TPT） | 4.1% |
| Gram矩阵正交（O-TPT） | 3.4% |
| 最小角距离最大化 | 3.8% |

Gram矩阵正交是最有效的约束方式。

**不同骨干网络上的表现**：

| 骨干 | TPT ECE | O-TPT ECE | 改善 |
|------|:------:|:--------:|:---:|
| ViT-B/16 | 12.8% | 3.4% | -73% |
| ViT-L/14 | 9.2% | 2.8% | -70% |
| ResNet-50 | 14.1% | 4.5% | -68% |

在所有骨干上一致有效。

## 五、局限性与不足

1. **继承TPT的速度问题**：仍需64x前向+反向传播
2. **正交假设的合理性**：完全正交意味着类别之间完全独立——但真实类别有层次结构（如"金毛"和"拉布拉多"应该有一定相似性），完全正交可能过于严格
3. **$\lambda$ 敏感**：不同数据集的最优 $\lambda$ 不同，需要调节
4. **类别数限制**：当类别数 $C$ 远大于嵌入维度 $d$ 时（$C >> d$），数学上不可能实现完全正交（鸽巢原理），此时正交约束会退化
5. **仅在分类任务上验证**：对检测、分割等密集预测任务的效果未知

## 六、核心价值

O-TPT将校准研究从"提高平均分离度"（C-TPT）推进到"保证最差情况分离度"（正交约束），在医学等安全关键领域具有重要价值：

> 在临床诊断中，一个校准良好的模型说"70%可能是恶性"比一个过度自信的模型说"99%是恶性"（实际可能只有50%准确率）更有价值。
